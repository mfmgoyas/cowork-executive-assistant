## Como usar este guia

1. **Antes de redigir**: identifique o destinatário, contexto e canal
2. **Escolha o idioma**: use a tabela de regra de idioma acima
3. **Aplique o tom correto**: Slack curto = direto; email stakeholder = estruturado; feedback = argumentado
4. **Revise contra anti-padrões**: elimine qualquer frase que você nunca escreveria
5. **Na dúvida**: menos é mais. Mensagens curtas e diretas são quase sempre melhores.
